В течение года, начиная с 15 января, в музее Современного Искусства будет проводится цикл из четырех выставок работ Хироши Сугимото.

Каждая из четырех выставок будет представлять одну из четырех тем работ художника: "Наука", "Архитектура", "История" и "Религия". Таким образом, посетители смогут увидеть полную панораму творчества Хироши Сугимото.

Часы работы: 10:00 - 18:00 (после 17:30 посетители в музей не допускаются).
