Морская биология

Состав морской воды

В морской воде можно найти почти все, включая растворенные вещества из земной коры, а также вещества, производимые организмами. Наиболее важными характеристиками морской воды, которые влияют на формы жизни, являются соленость, температура, растворенные газы (в основном кислород и углекислый газ), питательные вещества и рН. Эти характеристики оказывают различное влияние на морские организмы.
