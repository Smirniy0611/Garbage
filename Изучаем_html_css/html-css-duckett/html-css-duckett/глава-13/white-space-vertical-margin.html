Moog

Синтезаторы Moog были созданы доктором Робертом Мугом из компании Moog Music. Самыми популярными моделями были: Moog Modular, Minimoog, Micromoog, Moog Rogue и Moog Source.

ARP

Корпорация ARP Instruments, созданная Аланом Пералманом, была основным конкурентом компании Moog в 1970-x годах. Самыми популярными моделями были: Arp 2600 и ARP Odyssey.

Sequential Circuits

Корпорация Sequential Circuits, основанная Дэйвом Смитом, сыграла решающую роль в создании стандарта MIDI. Самыми популярными моделями были: Prophet 5, Prophet 600 и Pro-One.
