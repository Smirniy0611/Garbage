Фотография Описание Цена
Родес-пиано — это разновидность электромеханического пианино. Оно было изобретено Гарольдом Родесом в пятидесятые годы и позднее выпускалось в ряде вариаций, сначала в сотрудничестве с компанией Fender, а после 1965 года — компанией CBS. В этом инструменте используется похожая на фортепианную клавиатура с молоточками, которые ударяют по небольшим металлическим вибраторам, после чего сигнал усиливается электромагнитными усилителями. 240000 рублей
Электрическое пианино Вурлитцера было создано компанией Рудольфа Вурлитцера. Сама компания называла этот инструмент не «электрическим пианино», а «электронным пианино» и использовала это название в качестве товарного знака в течение всего времени производства. В этом инструменте используется похожая на фортепианную клавиатура с молоточками, которые ударяют по небольшим металлическим вибраторам, после чего сигнал усиливается электромагнитными усилителями. 87000 рублей
Клавинет — это «электрифицированная» версия клавикорда, выпускаемая компанией Hohner. Для ударения молоточком по струне каждая клавиша использует резиновый наконечник. Отличительное яркое стаккато данного инструмента часто сравнивают со звуком электрической гитары. На протяжении долгих лет производились различные модели, в том числе: I, II, L, C, D6 и E7. 46000 рублей
