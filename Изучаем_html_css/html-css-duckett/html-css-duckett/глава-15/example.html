Что такое Veloheld?

16 апреля 2011

Велосипеды Veloheld сочетают в себе минималистский дизайн и превосходное качество. Лишенная чрезмерных украшений и компонентов переключения передач линейка продуктов Veloheld радует нас своей красотой и простотой. Черный и белый велосипеды оснащены короткой колесной базой, одной передачей и узким рулем.

Для тех, кто хочет собрать велосипед самостоятельно, компания Veloheld предлагает рамы трех размеров и двух цветов, которые являются прекрасной отправной точкой. Далее...
